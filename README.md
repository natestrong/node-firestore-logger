###The Firestore Logger is a CLI to watch collections in Firestore.
It supports:
- [x] Watch multiple collections
- [x] Pass in simple or complex queries using Firestore's syntax: ```("isCool", "==", true)```
- [x] Watch Collection groups

Coming soon:
